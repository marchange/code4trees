Vom Zip zum Setzling

Drei Schritte, keine Haken. Dein Code bleibt deiner — wir prüfen nur, dass es sich um ein echtes Projekt handelt.

git add .

Projekt packen

Zippe dein abgeschlossenes Projekt — Quellcode plus kurze README, damit wir sehen, worum es geht.

git commit

Hochladen

Lade das Zip über das Formular hoch. Wir schauen kurz drüber, dass es ein echtes Projekt ist — mehr nicht.

git push --tree

Baum wird gepflanzt

Für jede angenommene Einreichung pflanzen wir mit unseren Partnern einen Baum und tragen ihn ins Pflanz-Log ein.

Projekt einreichen
